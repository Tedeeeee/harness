# gpt

LLM, 하네스, 에이전트 구조를 직접 구현하고 학습한 내용을 정리하는 프로젝트입니다.

이 저장소는 특히 `Claude Code`와 비슷한 형태의 하네스를 클린룸 방식으로 천천히 만들어보는 것을 목표로 합니다.  
한 번에 큰 시스템을 만드는 대신, 가장 작은 실행 루프부터 만들고 단계별 문서를 남기며 확장합니다.

## 현재 구현 단계

현재는 `02. 스트리밍 출력 추가` 단계까지 진행했습니다.

이 단계에서 가능한 것:

- CLI에서 프롬프트를 입력받을 수 있습니다.
- Anthropic API 키를 환경 변수에서 읽습니다.
- Anthropic API에 요청을 보낼 수 있습니다.
- 응답 텍스트를 스트리밍 방식으로 바로 출력합니다.
- 프롬프트 누락, API 키 누락을 오류로 안내합니다.

이미 확인한 것:

- TypeScript 빌드 성공
- 사용법 출력 확인
- API 키 누락 오류 확인
- 실제 Anthropic 응답 출력 확인
- 스트리밍 출력 흐름 테스트 통과

아직 없는 기능:

- REPL 루프
- 대화 이력
- 도구 호출
- 세션 저장
- 멀티 에이전트

## 실행 방법

### 1. 의존성 설치

```bash
npm install
```

### 2. 환경 변수 설정

`.env.example` 파일을 참고해서 프로젝트 루트에 `.env` 파일을 만들고 아래 값을 설정합니다.

```env
ANTHROPIC_API_KEY=your_anthropic_api_key_here
ANTHROPIC_MODEL=claude-sonnet-4-20250514
```

### 3. 개발 모드 실행

```bash
npm run dev -- "안녕하세요. 두 문장으로 짧게 자기소개를 해주세요."
```

### 4. 빌드 후 실행

```bash
npm run build
npm run start -- "안녕하세요. 두 문장으로 짧게 자기소개를 해주세요."
```

### 5. 테스트 실행

```bash
npm test
```
