# 문서 가이드

`docs/`는 하네스의 live operational state가 들어가는 곳입니다.

전체 설명과 운영 규칙은 루트 [README.md](/C:/study/harness/README.md) 를 기준으로 봅니다.

주요 live 영역:

- requirements
- visual-analysis
- architecture
- interview
- plans
- implementation
- verification

`visual-analysis/`는 이제 PDF/이미지뿐 아니라 프로토타입 코드 입력도 다룹니다. 이 문서들은 최종 구현 계약이 아니라 requirements 승격 전 분석 결과입니다.
