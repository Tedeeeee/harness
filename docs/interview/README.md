# Interview Directory

This directory stores development decisions confirmed through the planning interview.

## Purpose

These documents exist so planning can be based on explicit decisions instead of hidden assumptions.

## Main File

- `development-interview-decisions.md`

## Rules

- Record one decision topic at a time
- Capture the user's selected option or custom direction
- Use `confirmed` only for explicit approval or direct answers
- If AI first raised a missing item and the user approved it, record that clearly
