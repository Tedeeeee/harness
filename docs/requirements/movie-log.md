# 개인 영화 감상 기록 앱

## 목표

영화를 본 뒤 감상을 기록하고, 기록한 영화 목록을 쉽게 조회·관리할 수 있는 1인용 웹 앱을 만든다.

## 문제

영화를 보고 나서 별점과 감상을 정리해둘 개인 공간이 없다. 메모장이나 노션으로는 검색·정렬이 불편하다.

## 사용자

- 본인 (1인용, 로그인 없음)

## 핵심 기능

- 감상 기록 작성: 영화 제목, 별점(1~5), 한줄평, 상세 리뷰, 감상 날짜
- 감상 목록 조회: 전체 목록 보기
- 필터링: 별점 범위, 감상 날짜 범위
- 정렬: 최신순, 별점순, 제목순

## 범위 밖

- 외부 영화 검색 API (TMDB 등)
- 위시리스트 / 보고 싶은 영화 관리
- 통계 / 차트 (월별 감상 수, 장르별 분포 등)
- 사용자 인증 / 로그인
- 클라우드 배포

## 제약사항

- 기술 스택: Next.js + SQLite
- 로컬 환경에서만 실행
- 반응형 웹으로 모바일 브라우저 대응

## 성공 기준

- 새 감상 기록을 작성하고 저장할 수 있다
- 저장된 기록을 목록에서 확인할 수 있다
- 별점·날짜로 필터링, 최신순·별점순·제목순 정렬이 동작한다
- 기존 기록을 수정·삭제할 수 있다
- 모바일 브라우저에서 레이아웃이 깨지지 않는다
